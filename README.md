### ⏳ Installation

_Below is an example of how you install our project locally._

1. Clone the repo
   ```sh
   git clone https://github.com/seeewd/oss_steamtracker.git
   ```
2. Specify NVM version
   ```sh
   nvm use 16.10.0
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start website
   ```sh
   npm start
   ```
